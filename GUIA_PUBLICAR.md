# Publicar la web gratis

## Opcion mas facil: GitHub Pages

1. Crea una cuenta en GitHub si no la tienes.
2. Crea un repositorio nuevo, por ejemplo `mi-web`.
3. Sube el archivo `index.html`.
4. En GitHub entra en `Settings > Pages`.
5. En `Build and deployment`, elige:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` y carpeta `/root`
6. Guarda los cambios.
7. Tu web quedara publicada en una URL parecida a:
   - `https://tuusuario.github.io/mi-web/`

## Si quieres una URL gratis mas simple

Puedes usar tambien:

- Netlify: te da una URL tipo `nombre.netlify.app`
- Cloudflare Pages: te da una URL tipo `nombre.pages.dev`
- Vercel: te da una URL tipo `nombre.vercel.app`

## Importante sobre el dominio

- Un subdominio gratis si existe.
- Un dominio propio como `tunombre.com` normalmente no es gratis.
- Lo que si es gratis en muchos hostings es conectar un dominio que ya hayas comprado.

## Para salir en Google

- La web tiene que estar publicada publicamente.
- Tener `title` y descripcion ayuda.
- Debe verse bien en movil.
- Luego conviene darla de alta en Google Search Console y pedir indexacion.

## Archivo importante

El archivo de entrada para publicar es `index.html`.
